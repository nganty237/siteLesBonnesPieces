Projet web pédagogique permettant d'afficher, noter et commenter des pièces automobiles à partir d'une API simulée avec `json-server`.
Dossier : `1Front-End`

 Fichiers principaux :
`index.html` : page d’accueil avec les pièces affichées
`pieces.js` : affichage dynamique à partir des données JSON
`avis.js` : gestion des avis/commentaires
`styles.css` : mise en forme de la page


Dossier : `api-http`

# Technologies :
- [json-server](https://github.com/typicode/json-server) : simule une API REST locale

# Installation :

Ouvre un terminal dans `api-http/` puis exécute :

```bash
npm install
npm start //pour lancer le serveur
